Quickbooks Automatic Error Remover

While working at bill.com, there would be times where thousands of unnecessary errors would pop up for accounting software after data has tried to be sent to and from bill.com.

An example of this would be when the books are closed and a list item is merged incorrectly, causing errors on records that that you donâ€™t want to sync to your accounting system.

Unlike other cloud based accounting systems, the Quickbooks desktop bill.com integration does not have built in support for clearing errors in bulk. Meaning that clearing the errors could end up being a manual task that takes dozens of minutes.

This is a chrome extension built to automatically clear sync errors for Quickbooks desktop in bill.com.
